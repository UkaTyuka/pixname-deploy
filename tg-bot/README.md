# Telegram Bot

Telegram bot that implements the core business logic of the application.  
Built with Python using the [`python-telegram-bot`](https://github.com/python-telegram-bot/python-telegram-bot) library.

### Features and Commands:

- `/help` – displays a list of available commands and usage instructions  
- `/about_us` – provides information about the bot and its purpose  
- `/generate_text` – generates a text description based on an uploaded image  

The bot serves as the main user interface, handling user interactions and forwarding requests to backend services such as the ML core and database.
